# 🚀 快速开始指南

## 立即运行游戏

### 1. 启动开发服务器

```bash
npm start
```

服务器将在 `http://localhost:8888` 启动，浏览器会自动打开。

### 2. 游戏操作

#### 选择关卡
- 在主界面点击任意关卡卡片开始游戏
- 目前有3个关卡可供选择

#### 玩游戏
- **左键点击**方块：循环切换颜色（空白 → 颜色1 → 颜色2 → 空白）
- **右键点击**方块：清空方块
- 观察网格边缘的提示来推理正确的图案
- 填满所有方块后点击"检查答案"

#### 提示说明
- 行提示（网格左侧）：显示每行从左到右的颜色和连续方块数量
- 列提示（网格上方）：显示每列从上到下的颜色和连续方块数量
- 例如：`🔴2 ⚪1 🔴2` 表示 2个红色、1个白色、2个红色

#### 辅助功能
- **检查答案**：验证你的答案是否正确
- **重置**：清空所有填色重新开始
- **提示**：自动填充一个正确的格子
- **返回关卡选择**：返回主菜单选择其他关卡

## 游戏技巧

### 初学者建议
1. **从简单的提示开始**：找到只有一种可能的行或列
2. **标记确定的格子**：先填充你确定的格子
3. **交叉验证**：用行提示和列提示互相验证
4. **使用排除法**：有时通过排除不可能的选项来找答案

### Nonogram 解题策略
1. **寻找完整行/列**：如果提示的数字总和等于网格大小，那么整行/列都是该颜色
2. **边缘推理**：从边缘开始，根据提示填充一定正确的格子
3. **重叠技巧**：如果某个颜色块在所有可能的排列中都会出现在某个位置，那么该格子一定是那个颜色

## 常见问题

### Q: 如何知道答案是否正确？
A: 填满所有格子后点击"检查答案"按钮，系统会自动验证。正确会显示绿色成功消息，错误会显示红色提示。

### Q: 卡住了怎么办？
A: 可以使用"提示"按钮，系统会自动帮你填充一个正确的格子。

### Q: 可以撤销操作吗？
A: 当前版本暂不支持撤销。但你可以随时点击方块重新填色，或使用"重置"按钮清空重来。

### Q: 为什么有些提示有多个颜色和数字？
A: 这表示该行/列有多段不同颜色的连续方块。例如 `🔴2 ⚪1 🔴1` 表示：2个红色、接着1个白色、接着1个红色。

### Q: 如何添加更多关卡？
A: 参考 `README.md` 中的"添加新关卡"部分，在 `src/app/data/levels.ts` 中添加新的关卡数据。
